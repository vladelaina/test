## [🌟功能特点](https://vladelaina.github.io/Catime/#core-features)

- **极简设计**：透明界面，点击穿透，自由拖拽和调整大小，无复杂图形设置界面，使用时不会感觉是额外的应用程序在运行
- **灵活计时**：快速预设，自定义时间输入，当前时间显示（12/24小时制），倒计时，正计时，番茄钟
- **超时动作**：显示消息，显示当前时间，正计时，锁屏，打开文件/应用程序，打开网站，关机，重启
- **丰富自定义**：13种字体，15种预设颜色，支持自定义颜色值，颜色调色板选择，实时预览

## [📑用户指南](https://vladelaina.github.io/Catime/guide)

- **左键点击托盘图标**：设置**时间**
- **右键点击托盘图标**：设置**菜单**
- **编辑模式**：拖动调整位置，滚轮调整大小，🖱️右键退出编辑模式

> 配置文件保存在 `C:\Users\<username>\AppData\Local\Catime\config.ini`




## 🛠️ [从源代码构建](https://www.bilibili.com/video/BV1H97LzVEee)

### 1. 克隆:

```
git clone git@github.com:vladelaina/Catime.git
cd Catime
```

### 2. 工具 (MinGW, CMake)


#### <img src="../Images/linux.svg"  height="25" />Linux

- <img src="../Images/ubuntu.svg"  height="25" />Ubuntu
   ```bash
   sudo apt update && sudo apt install -y mingw-w64 cmake
   ```
- <img src="../Images/archlinux.svg"  height="25" />Arch
  ```bash
  sudo pacman -Syu --noconfirm mingw-w64 cmake
  ```

#### <img src="../Images/windows.svg"  height="25" />Windows
  
  1. 准备工具

  | 工具       | 描述            | 推荐版本格式                                    | 下载链接                                                                     |
  | ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
  | **MinGW**  | GCC编译器           | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)  |
  | **CMake**  | 构建工具             | `cmake-<version>-windows-x86_64.msi`                         | [CMake](https://cmake.org/download/)                                            |

  <details>
    <summary>2. 安装工具</summary>

  #### 📦 2.1 安装 [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)

  1. **将MinGW解压到特定位置**
     示例：将 `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` 解压到：

     ```
     C:\mingw64
     ```

  2. **配置系统环境变量PATH**

     * 打开：`控制面板 → 系统 → 高级系统设置 → 环境变量`
     * 在**系统变量**中找到 `Path`，点击"编辑"
     * 添加以下路径：

       ```
       C:\mingw64\bin
       ```

  3. **验证安装**

     打开命令提示符（Win + R → 输入 `cmd` → 回车），输入：

     ```bash
     gcc --version
     ```

     如果成功显示版本号，MinGW安装完成 ✅

  #### 📦 2.2 安装 [CMake](https://cmake.org/download/)

  1. 下载并运行 `cmake-<version>-windows-x86_64.msi` 安装
  2. 安装时选择"为所有用户添加CMake到系统PATH"或"为当前用户添加"
  3. 或者手动将CMake安装目录添加到PATH

  </details>


### 3. 验证工具
  <details>
    <summary></summary>


  #### ✅ 3.1 验证gcc

  ```bash
  gcc --version
  ```

  #### ✅ 3.2 验证CMake

  ```bash
  cmake --version
  ```

  如果都正确显示版本号，则工具配置成功 🎉
  </details>





### 4. 使用CMake构建

#### Windows

使用提供的构建脚本：

```bash
build.bat             # Release构建
build.bat Debug       # Debug构建
```

或手动使用CMake构建：

```bash
mkdir build
cd build
cmake .. -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release
cmake --build .
```

#### Linux（交叉编译）

使用提供的构建脚本：

```bash
./build.sh            # 在'build'目录下Release构建
./build.sh Debug      # 在'build'目录下Debug构建
./build.sh Release ./dist  # 在'dist'目录下Release构建
```

或手动使用CMake构建：

```bash
mkdir build
cd build
cmake .. -DCMAKE_TOOLCHAIN_FILE=../mingw-w64-toolchain.cmake -DCMAKE_BUILD_TYPE=Release
cmake --build .
```

#### 常用CMake命令

```bash
cmake --build . --config Release  # 编译项目
cmake --build . --target clean    # 清理构建文件
```
