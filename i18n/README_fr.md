## [🌟 Fonctionnalités](https://vladelaina.github.io/Catime/#core-features)
- **Design minimaliste** : Interface transparente, clic traversant, redimensionnement libre, pas d'interface graphique complexe
- **Flexibilité temporelle** : Présélections rapides, entrée personnalisée, formats 12/24h, compte à rebours/minuterie, Pomodoro
- **Actions programmables** : Notification, affichage horaire, minuterie, verrouillage écran, lancement fichier/application/site web, extinction/redémarrage
- **Personnalisation avancée** : 13 polices, 15 couleurs prédéfinies, palette personnalisable, prévisualisation en temps réel

## [📑 Guide utilisateur](https://vladelaina.github.io/Catime/guide)
- **Clic gauche sur l'icône** : Définir le **temps**
- **Clic droit sur l'icône** : Accéder au **menu**
- **Mode édition** : Déplacement par glisser, redimensionnement par molette, 🖱️clic droit pour quitter
> Configuration sauvegardée dans `C:\Users\<utilisateur>\AppData\Local\Catime\config.ini`
## 🛠️ [Compilation depuis les sources](https://www.bilibili.com/video/BV1H97LzVEee)
### 1. Cloner :
```
git clone git@github.com:vladelaina/Catime.git
cd Catime
```
### 2. Outils requis (MinGW, CMake)


#### <img src="../Images/linux.svg"  height="25" />Linux

- <img src="../Images/ubuntu.svg"  height="25" />Ubuntu
   ```bash
   sudo apt update && sudo apt install -y mingw-w64 cmake
   ```
- <img src="../Images/archlinux.svg"  height="25" />Arch
  ```bash
  sudo pacman -Syu --noconfirm mingw-w64 cmake
  ```
#### <img src="../Images/windows.svg"  height="25" />Windows

1. Préparer les outils

| Outil       | Description            | Format de version recommandé                                    | Lien de téléchargement                                                                     |
| ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **MinGW**  | Compilateur GCC           | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)  |
| **CMake**  | Outil de construction             | `cmake-<version>-windows-x86_64.msi`                         | [CMake](https://cmake.org/download/)                                            |

<details>
<summary>2. Installer les outils</summary>

#### 📦 2.1 Installer [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)

1. **Extraire MinGW à un emplacement spécifique**
   Exemple : extraire `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` vers :
     ```
     C:\mingw64
     ```
     
2. **Configurer la variable d'environnement PATH**
* Ouvrir : `Panneau de configuration → Système → Paramètres système avancés → Variables d'environnement`
* Trouver `Path` dans les **variables système**, cliquer sur "Modifier"
* Ajouter le chemin :

  ```
  C:\mingw64\bin
  ```

3. **Vérifier l'installation**
Ouvrir l'invite de commandes (Win + R → taper `cmd` → Entrer), puis taper :

     ```bash
     gcc --version
     ```
      Si la version s'affiche correctement, l'installation est réussie ✅

#### 📦 2.2 Installer  [CMake](https://cmake.org/download/)

  1. Exécuter cmake-<version>-windows-x86_64.msi pour installer
  2. Le PATH sera configuré automatiquement durant l'installation (sinon, ajouter manuellement le répertoire bin de CMake au PATH)

  </details>
  
### 3.Vérifier les outils
  <details>
    <summary></summary>


  #### ✅ 3.1 Vérifier gcc

  ```bash
  gcc --version
  ```

  #### ✅ 3.2 Vérifier CMake

  ```bash
  CMake --version
  ```

  Si les versions s'affichent correctement, la configuration est réussie 🎉
  </details>

### 4.Construire avec CMake

Dans le répertoire racine du projet, ouvrir l'invite de commandes et utiliser :

```bash
CMake         # Compiler le projet
CMake run     # Compiler et exécuter le projet
CMake clean   # Nettoyer les fichiers de construction
```
