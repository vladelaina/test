<p align="center" style="font-size: 18px; margin-bottom: 30px;">
  🎉 Keine Installation oder Einrichtung erforderlich - ein Doppelklick genügt, um loszulegen.<br/>
  🌸 Saubere, elegante Oberfläche mit reibungsloser Bedienung.<br/>
  🚀 Leistungsstarke Tastaturkürzel für blitzschnelle Produktivität.
</p>

<p align="center" style="font-size: 16px; color: #666; margin-bottom: 20px;">
  Ein ultraleichtes Windows-Countdown-Tool mit Pomodoro-Funktionalität, <br/>
  das sich durch eine transparente Oberfläche und umfangreiche Anpassungsoptionen auszeichnet. <br/>
  Es benötigt nur wenige MB Speicher und fast keine CPU-Ressourcen und hilft Ihnen auf elegante Weise, <br/>
  die Zeit unter Windows zu verwalten.
</p>

## [🌟Eigenschaften](https://vladelaina.github.io/Catime/#core-features)

- **Minimalistisches Design**: Transparente Oberfläche, Durchklickmöglichkeit, freies Ziehen und Größenänderung, keine komplexe grafische Einstellungsoberfläche, fühlt sich nicht wie eine zusätzliche Anwendung an
- **Flexible Zeitplanung**: Schnellvoreinstellungen, benutzerdefinierte Zeiteingabe, aktuelle Zeitanzeige (12/24-Stunden-Format), Countdown, Count-up, Pomodoro-Uhr
- **Timeout-Aktionen**: Nachrichten anzeigen, aktuelle Uhrzeit anzeigen, hochzählen, Bildschirm sperren, Dateien/Anwendungen öffnen, Websites öffnen, herunterfahren, neu starten
- **Umfangreiche Anpassung**: 13 Schriftarten, 15 voreingestellte Farben, Unterstützung für benutzerdefinierte Farbwerte, Farbpalettenauswahl, Echtzeit-Vorschau

## [📑Benutzerhandbuch](https://vladelaina.github.io/Catime/guide)

- **Linksklick auf das Tray-Symbol**: eingestellte **Zeit**
- **Rechtsklick auf das Tray-Symbol**: **Festmenü**
- **Bearbeitungsmodus**: Ziehen zum Anpassen der Position, Scrollrad zum Ändern der Größe, 🖱️ Rechtsklick zum Verlassen des Bearbeitungsmodus

> Konfigurationsdatei gespeichert unter `C:\Users\<username>\AppData\Local\Catime\config.ini`




## 🛠️ [Bauen von der Quelle aus](https://www.bilibili.com/video/BV1H97LzVEee)

### 1. Klonen:

```
git clone git@github.com:vladelaina/Catime.git
cd Catime
```

### 2. Tools (MinGW, CMake)


#### <img src="../Images/linux.svg"  height="25" />Linux

- <img src="../Images/ubuntu.svg"  height="25" />Ubuntu
   ```bash
   sudo apt update && sudo apt install -y mingw-w64 && curl -fsSL https://CMake.io/shget.text | bash
   ```
- <img src="../Images/archlinux.svg"  height="25" />Arch
  ```bash
  sudo pacman -Syu --noconfirm mingw-w64 CMake
  ```

#### <img src="../Images/windows.svg"  height="25" />Windows
  
  1. Tools vorbereiten

  | Tool       | Beschreibung           | Empfohlene Version Format                                   | Link herunterladen                                                                   |
  | ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
  | **MinGW**  | GCC Compiler           | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)  |
  | **CMake**  | Build Tool             | `cmake-<version>-windows-x86_64.msi`                                  | [CMake](https://cmake.org/download/)                      |

  <details>
    <summary>2. Tools installieren</summary>

  #### 📦 2.1 Installieren Sie [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)

  1. **MinGW an einen bestimmten Ort extrahieren**
     Beispiel: Extrahieren Sie `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` nach:

     ```
     C:\mingw64
     ```

  2. **Konfigurieren Sie die Systemumgebungsvariable PATH**

     * Öffnen Sie: „Systemsteuerung → System → Erweiterte Systemeinstellungen → Umgebungsvariablen“.
     * Suchen Sie „Pfad“ in **Systemvariablen**, klicken Sie auf „Bearbeiten“.
     * Fügen Sie den folgenden Pfad hinzu:

       ```
       C:\mingw64\bin
       ```

  3. **Überprüfung der Installation**

     Öffnen Sie die Eingabeaufforderung (Win + R → „cmd“ → Enter), geben Sie ein:

     ```bash
     gcc --version
     ```

     Wenn die Versionsnummer erfolgreich angezeigt wird, ist die Installation von MinGW abgeschlossen. ✅

  #### 📦 2.2 Installieren [CMake](https://cmake.org/download/)

  1. Führen Sie „CMake-v<Version>-win64.exe“ zur Installation aus
  2. PATH wird während der Installation automatisch konfiguriert (falls nicht, fügen Sie das `bin`-Verzeichnis der CMake-Installation manuell hinzu)

  </details>


### 3. Tools überprüfen
  <details>
    <summary></summary>


  #### ✅ 3.1 Überprüfen Sie gcc

  ```bash
  gcc --version
  ```

  #### ✅ 3.2 Überprüfen Sie CMake

  ```bash
  CMake --version
  ```

  Wenn alle Versionsnummern korrekt angezeigt werden, ist die Konfiguration des Tools erfolgreich. 🎉
  </details>





### 4. Erstellen mit CMake

Öffnen Sie eine Eingabeaufforderung im Stammverzeichnis des Projekts und verwenden Sie folgende Befehle:

```bash
CMake         # Kompilieren Sie das Projekt
CMake run     # Kompilieren und Ausführen des Projekts
CMake clean   # Saubere Build-Artefakte
```
