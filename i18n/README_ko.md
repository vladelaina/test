## [🌟기능 특징](https://vladelaina.github.io/Catime/#core-features)

- **초간단 디자인**: 투명 인터페이스, 클릭 투과, 자유로운 드래그 및 크기 조정, 복잡한 그래픽 설정 인터페이스 없음, 실행 중인 별도의 애플리케이션처럼 느껴지지 않음
- **유연한 타이머**: 빠른 프리셋, 사용자 정의 시간 입력, 현재 시간 표시(12/24시간제), 카운트다운, 카운트업, 토마토 타이머
- **타임아웃 동작**: 메시지 표시, 현재 시간 표시, 카운트업, 화면 잠금, 파일/애플리케이션 열기, 웹사이트 열기, 종료, 재부팅
- **풍부한 사용자 정의**: 13가지 폰트, 15가지 프리셋 색상, 사용자 정의 색상 값 지원, 색상 팔레트 선택, 실시간 미리보기

## [📑사용자 가이드](https://vladelaina.github.io/Catime/guide)

- **트레이 아이콘 왼쪽 클릭**: **시간** 설정
- **트레이 아이콘 오른쪽 클릭**: **메뉴** 설정
- **편집 모드**: 드래그로 위치 조정, 휠로 크기 조정, 🖱️오른쪽 클릭으로 편집 모드 종료

> 설정 파일은 `C:\Users\<username>\AppData\Local\Catime\config.ini`에 저장됨

## 🛠️ [소스 코드에서 빌드](https://www.bilibili.com/video/BV1H97LzVEee)

### 1. 클론:

```
git clone git@github.com:vladelaina/Catime.git
cd Catime
```

### 2. 도구 (MinGW, CMake)

#### <img src="../Images/linux.svg"  height="25" />Linux

- <img src="../Images/ubuntu.svg"  height="25" />Ubuntu
   ```bash
   sudo apt update && sudo apt install -y mingw-w64 && curl -fsSL https://CMake.io/shget.text | bash
   ```
- <img src="../Images/archlinux.svg"  height="25" />Arch
  ```bash
  sudo pacman -Syu --noconfirm mingw-w64 CMake
  ```

#### <img src="../Images/windows.svg"  height="25" />Windows
  
  1. 도구 준비

  | 도구       | 설명            | 추천 버전 형식                                    | 다운로드 링크                                                                     |
  | ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
  | **MinGW**  | GCC 컴파일러           | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)  |
  | **CMake**  | 빌드 도구             | `cmake-<version>-windows-x86_64.msi`                                  | [CMake](https://cmake.org/download/)                      |

  <details>
    <summary>2. 도구 설치</summary>

  #### 📦 2.1 [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) 설치

  1. **MinGW를 특정 위치에 압축 해제**
     예: `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z`를 다음 위치에 압축 해제:

     ```
     C:\mingw64
     ```

  2. **시스템 환경 변수 PATH 설정**

     * 열기: `제어판 → 시스템 → 고급 시스템 설정 → 환경 변수`
     * **시스템 변수**에서 `Path`를 찾아 "편집" 클릭
     * 다음 경로 추가:

       ```
       C:\mingw64\bin
       ```

  3. **설치 확인**

     명령 프롬프트 열기(Win + R → `cmd` 입력 → Enter), 입력:

     ```bash
     gcc --version
     ```

     버전 번호가 성공적으로 표시되면 MinGW 설치 완료 ✅

  #### 📦 2.2 [CMake](https://cmake.org/download/) 설치

  1. `cmake-<version>-windows-x86_64.msi` 실행하여 설치
  2. 설치 과정에서 PATH가 자동 설정됨(설정되지 않은 경우, CMake 설치 디렉토리의 `bin` 디렉토리를 PATH에 수동 추가)

  </details>

### 3. 도구 확인
  <details>
    <summary></summary>

  #### ✅ 3.1 gcc 확인

  ```bash
  gcc --version
  ```

  #### ✅ 3.2 CMake 확인

  ```bash
  CMake --version
  ```

  두 도구 모두 버전 번호가 정상적으로 표시되면 도구 설정 성공 🎉
  </details>

### 4. CMake로 빌드

프로젝트 루트 디렉토리에서 명령 프롬프트를 열고 다음 명령을 사용:

```bash
CMake         # 프로젝트 컴파일
CMake run     # 컴파일 및 실행
CMake clean   # 빌드 파일 정리
```
